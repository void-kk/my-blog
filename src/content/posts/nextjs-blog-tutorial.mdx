# Next.js로 블로그 만들기

이 블로그는 Next.js를 사용하여 만들었습니다. 오늘은 어떻게 이 블로그를 만들었는지 공유해보려고 합니다.

## 사용한 기술 스택

- **Next.js**: React 기반 프레임워크
- **TypeScript**: 타입 안정성
- **Tailwind CSS**: 스타일링
- **MDX**: Markdown + React 컴포넌트
- **GitHub Pages**: 호스팅

## 주요 기능

### 1. MDX 지원

MDX를 사용하면 Markdown 안에서 React 컴포넌트를 사용할 수 있습니다. 이를 통해 더욱 풍부한 콘텐츠를 작성할 수 있습니다.

```javascript
// MDX 예시
import { CustomComponent } from './components'

# 제목

일반 마크다운 텍스트

<CustomComponent prop="value" />
```

### 2. 다크 모드

`next-themes` 라이브러리를 사용하여 다크 모드를 구현했습니다. 시스템 설정에 따라 자동으로 테마가 변경되며, 사용자가 직접 토글할 수도 있습니다.

### 3. 정적 내보내기

GitHub Pages에 배포하기 위해 정적 내보내기(`output: 'export'`)를 사용했습니다. 이를 통해 빌드 시 모든 페이지가 정적 HTML로 생성됩니다.

## 마무리

Next.js를 사용하면 쉽게 블로그를 만들 수 있습니다. 특히 MDX를 활용하면 기술 블로그에 적합한 풍부한 콘텐츠를 작성할 수 있습니다.
